# The Time Sphere
### thetimesphere.com — Global Time Intelligence Platform

Live world clocks, time zones, UTC offsets, and DST status for every major city on Earth.

---

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage — city search + live world clock grid |
| `time.html` | Dynamic city page — `?city=london` etc. |
| `about.html` | About page |
| `contact.html` | Contact form (Formspree) |
| `privacy.html` | Privacy policy (required for AdSense) |
| `style.css` | Global styles |
| `app.js` | City data + live clock logic |
| `PRD.md` | Product Requirements Document |

## Setup

1. Clone this repo
2. Open `index.html` in a browser (or use Live Server in VS Code)
3. Update `contact.html` — replace `YOUR_FORMSPREE_ID` with your Formspree form ID
4. When AdSense is approved, replace `.ad-slot` divs with your ad unit code

## Deployment

Deployed via Railway connected to this GitHub repo.

## Tech Stack

- HTML5 + CSS3 + Vanilla JavaScript
- Google Fonts: Orbitron + Exo 2
- Formspree (contact form)
- Railway (hosting)
